# The-Last-Broadcast

Set during a fictional 1940s wartime conflict, the player takes control of a fighter pilot charged with navigating through hostile territory. Enemy forces have destroyed key radio towers, cutting off communication between allied bases. Your mission is to fly a dangerous route, defending yourself from enemy aircraft, to reconnect the broken broadcast network before the final message is lost.

The core game loop focuses on flying a fixed path, shooting down enemy targets, avoiding anti-aircraft fire, and repair damaged radio towers by targeting them with a short "signal pulse" rather than bullets.

As each radio tower is re-established, communication is restored between isolated allied units. As the player progresses, radio signals become clearer as voices, music and mission updates gradually return.
